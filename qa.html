SDLC vs STLC Understanding

SDLC — Software Development

  • Requirement Gathering
  • System Design
  • Implementation / Coding
  • Testing
  • Deployment
  • Maintenance

STLC — Software Testing

  • Requirement Analysis
  • Test Planning
  • Test Case Development
  • Test Environment Setup
  • Test Execution
  • Test Closure
